**Basic-excel-kpi-dashboard
Excel KPI Dashboard**

This project is an interactive Excel dashboard created using:

- Excel formulas
- Data validation dropdowns
- KPI cards
- Charts & graphs

** Features**
- Dynamic KPI cards
- Dropdown-based filtering
- Interactive charts
- Clean dashboard layout

**Tools Used**
- Microsoft Excel
